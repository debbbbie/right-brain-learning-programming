### 打油诗记忆 Markdown

> 一星斜 来二星粗，两弯拿走大引路
> 
> 中括小括是超链，带上叹号成图片
> 
> 加减星号是无序，数字加点才有序


### 编码表

|记忆内容|编码|备注|
|  ----  | ---- |---- |
文本|一星*斜* 来二星**粗**，两弯~~拿走~~大引路 | \* ** ~~ >
链接 & 图片| 中括小括是超链，带上叹号成图片  |`[text](url)` &  `![alt](url)`
列表|加减星号是无序，数字加点才有序

## Markdown 简明语法手册

|记忆内容|编码|备注|
|  :----:  | ---- |---- |
*\*斜体\**| 重要| *
**\*\*粗体\*\***| 更重要| **
~~\~\~删除线\~\~~~ | |~~
无序列表 * + - |  | 
有序列表 1. 2. ...|  | 数字和点
文字引用  |  | `>`
超链接  |  | `[text](url)`
图片  |  | `![alt](url)`
分级标题| |`# H1, ## H2, ### H3`
注脚[^1] | | `[^keyword]`
 行内代码块| | \`代码`
代码块|| 缩进或 ```
表格|| `Left__ | _Right` <br/>  `:----- | -----:`
